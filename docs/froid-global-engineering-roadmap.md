# FROID - Global Engineering Roadmap (MVP -> Escala Global)

## Sprint 0 (base técnica)
- [ ] Pipeline CI/CD (build, smoke test auth, rollback)
- [ ] Padrão de versionamento de API
- [ ] Observabilidade mínima (logs estruturados + health endpoints)

## Sprint 1 (UX/UI internacional)
- [ ] Design system (tokens, tipografia, componentes)
- [ ] Internacionalização (i18n): pt-BR, en-US, es-ES
- [ ] Acessibilidade WCAG 2.2 AA (teclado, contraste, labels)

## Sprint 2 (fluxos clínicos)
- [ ] Onboarding profissional (psicólogo/psiquiatra)
- [ ] Fluxo de sessão (início, consentimentos, encerramento)
- [ ] Fluxo de relatório com trilha de auditoria

## Sprint 3 (escala global)
- [ ] Timezone e locale por usuário
- [ ] Multi-tenant por clínica/país
- [ ] Catálogo de compliance por região (LGPD/GDPR)

## Sprint 4 (segurança e operação)
- [ ] Rotação de segredos automatizada
- [ ] Backups e restore testado
- [ ] SLO/SLA + alertas

## Critérios de aceite (todos os sprints)
- [ ] Smoke test register/login passa
- [ ] Sem erro crítico no log em 24h
- [ ] Documentação atualizada
